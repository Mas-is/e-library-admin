Ringkasan Sistem

E-Library Admin Panel adalah aplikasi front-end untuk mengelola buku fisik, anggota, peminjaman, pengembalian, laporan, dan pengaturan sistem.

  • Data disimpan di localStorage browser.
  • Website dapat dibuka di desktop dan handphone.
  • Admin mengonfirmasi pengembalian dari anggota.

Panduan Admin

  • Dashboard menampilkan total buku, anggota, peminjaman aktif, dan keterlambatan.
  • Data Buku digunakan untuk tambah, edit, hapus, cari, dan filter buku.
  • Data Kategori tersinkron ke form tambah buku.
  • Data Anggota digunakan untuk mengelola akun anggota.

Alur Transaksi

  • Peminjaman dari admin atau anggota akan mengurangi stok buku.
  • Anggota dapat mengajukan pengembalian.
  • Admin memproses pengembalian final pada menu Pengembalian.
  • Laporan dan notifikasi ikut terupdate otomatis.

Laporan & Export

  • Menu Laporan mendukung pencarian dan filter tanggal.
  • Data dapat dicetak melalui tombol Print.
  • Export Excel menghasilkan file .xls dari data yang sedang aktif.
  • Untuk reset demo, hapus localStorage browser.
